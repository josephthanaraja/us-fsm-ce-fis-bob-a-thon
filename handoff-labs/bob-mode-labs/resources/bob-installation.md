# Installing Bob

You will have been given a link to download Bob or provided the installation file.

## 1. Follow the installation steps for your operating system

### macOS

1. Open the downloaded `.dmg` file
2. Drag the Bob application to your Applications folder
3. Open Bob from your Applications folder
4. If you see a security warning, go to **System Preferences > Security & Privacy** and click **Open Anyway**

### Linux

1. Extract the downloaded archive:
   ```bash
   tar -xzf Bob-<version>.tar.gz
   ```

2. Navigate to the extracted directory:
   ```bash
   cd Bob-<version>
   ```

3. Run Bob:
   ```bash
   ./Bob
   ```

> **Note:** You might want to move the extracted Bob directory to a permanent location (such as `/opt/Bob` or `~/Applications/Bob`) and create a symbolic link or desktop entry for easier access.

### Windows

1. Run the downloaded `.exe` installer
2. Follow the installation wizard prompts
3. Choose your installation directory (default is recommended)
4. Complete the installation

## 2. Launch Bob

After installing, open Bob from your applications menu or desktop shortcut. Bob will open as a standalone application.

## 3. Authenticate

On first launch, you'll need to authenticate:

1. Bob will prompt you to sign in with your credentials
2. Follow the authentication flow in your browser
3. Return to Bob after successful authentication
