# Custom Handoff Labs for FIS Bob-a-thon

Four advanced labs demonstrating real-world Bob applications in CI/CD pipelines.

---

## Available Labs

### 1. Auto-Recovery and Self-Healing Lab
**File:** [`bob-auto-recovery-self-healing-lab-CUSTOM.md`](bob-auto-recovery-self-healing-lab-CUSTOM.md)  
**Duration:** ~45 minutes | **Level:** Intermediate to Advanced

Build automated error detection and recovery in Jenkins pipelines with self-healing build stages and intelligent dependency resolution.

**Prerequisites:** Jenkins pipeline knowledge, Maven familiarity

---

### 2. MCP Server Development Lab
**File:** [`bob-mcp-server-development-CUSTOM.md`](bob-mcp-server-development-CUSTOM.md)  
**Duration:** ~60 minutes | **Level:** Advanced

Create custom MCP servers for pipeline metrics, build history queries, and automated report generation.

**Prerequisites:** MCP familiarity, basic Node.js, Jenkins knowledge

---

### 3. OpenShift Deployment Lab
**File:** [`bob-openshift-deployment-lab-CUSTOM.md`](bob-openshift-deployment-lab-CUSTOM.md)  
**Duration:** ~60 minutes | **Level:** Advanced

Implement pre-deployment validation, automated container builds, deployment orchestration, and intelligent rollback automation.

**Prerequisites:** OpenShift cluster access, container/Kubernetes understanding

---

### 4. Bob Mode Builder Lab
**File:** [`bob-mode-labs/README.md`](bob-mode-labs/README.md)  
**Duration:** ~30 minutes to 2+ hours | **Level:** Beginner to Advanced

Create custom Bob modes tailored to your workflows. Learn to build simple modes (code review, documentation) and workflow modes (feature development, bug fixes) with reusable templates.

**Why it matters:** Custom modes are a key Bob differentiator, enabling 60-80% development acceleration and consistent code quality.

**Includes:** 30-min main guide, advanced topics, 5 example modes, templates, resources

**Prerequisites:** Basic Bob familiarity
