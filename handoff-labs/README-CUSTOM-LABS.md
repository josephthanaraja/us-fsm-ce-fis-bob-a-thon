# Bob Handoff Labs

Advanced labs demonstrating Bob's capabilities in CI/CD pipelines and custom development workflows.

---

## Available Labs

### 1. Auto-Recovery and Self-Healing
**File:** [`bob-auto-recovery-self-healing-lab.md`](bob-auto-recovery-self-healing-lab.md)
**Duration:** ~45 minutes | **Level:** Intermediate to Advanced

Automate error detection and recovery in Jenkins pipelines with self-healing build stages.

**Prerequisites:** Jenkins pipeline knowledge, Maven familiarity

---

### 2. OpenShift Deployment Automation
**File:** [`bob-openshift-deployment-lab.md`](bob-openshift-deployment-lab.md)
**Duration:** ~60 minutes | **Level:** Advanced

Automate pre-deployment validation, container builds, deployment orchestration, and intelligent rollback.

**Prerequisites:** OpenShift cluster access, container/Kubernetes understanding

---

### 3. Bob Mode Builder
**File:** [`bob-mode-labs/README.md`](bob-mode-labs/README.md)
**Duration:** ~30 minutes to 2+ hours | **Level:** Beginner to Advanced

Create custom Bob modes for your workflows. Includes 30-min guide, 5 examples, and templates.

**Key benefit:** Custom modes enable 60-80% development acceleration and consistent code quality.

**Prerequisites:** Basic Bob familiarity

---

### 4. Example MCP Servers
**File:** [`ibm-bob-example-mcp-servers-main/README.md`](ibm-bob-example-mcp-servers-main/README.md)
**Duration:** 30 min - 3+ hours | **Level:** Beginner to Advanced

Eight production-ready MCP server examples using FastMCP: calculator, file operations, web scraping, database operations, API integration, containerized execution.

**Prerequisites:** Python 3.8+, pip (Podman for Lab 07)

---

## Getting Started

1. **Choose a lab** based on your interest
2. **Review prerequisites** to ensure you have required access/tools
3. **Open the lab file** and follow step-by-step instructions

**Note:** Labs 1-2 use the FIS Bob-a-thon workshop environment (Jenkins, OpenShift, Bob CLI, Maven). Labs 3-4 are standalone and work in any environment.
